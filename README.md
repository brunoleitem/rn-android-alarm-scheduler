# rn-android-alarm-scheduler

Alarm utils for react native (Android)

## Installation

```sh
npm install rn-android-alarm-scheduler
```

## Usage

```js
import { multiply } from 'rn-android-alarm-scheduler';

// ...

const result = await multiply(3, 7);
```
